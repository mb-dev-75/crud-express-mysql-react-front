## How To Use

```bash
# Clone this repository
$ git clone https://github.com/mb-dev-75/crud-express-mysql-react-front

# Go into the repository
$ cd crud-express-mysql-react-front

# Install dependencies
$ yarn

# Run the app
$ yarn start
```
